# checklist-java-wildfly

Qué incluye
Funcionalidad:

-Checklist interactivo con casillas marcables (~30 items). El progreso se guarda automáticamente en tu navegador (localStorage), así que podés cerrar el archivo y retomar después.
-Widget de progreso en el sidebar que muestra X / Total y una barra que se llena a medida que avanzás.
-Sidebar fijo navegable con tabla de contenidos. Se resalta automáticamente la sección que estás viendo.
-Botón "copy" en cada bloque de código para pegar directo en PowerShell sin errores de espacios.
-Sección desplegable de troubleshooting con los 12 errores que encontramos y resolvimos durante la sesión.
-Botón "Reset" para empezar de cero si rehacés la instalación en otra máquina.
-Print-friendly — Ctrl+P genera un PDF razonable sin el sidebar.

Una nota práctica
Si vas a hacer esto en otra máquina más adelante, te conviene guardar el archivo HTML en algún lugar accesible (OneDrive, Drive, GitHub gist privado). El archivo es totalmente autocontenido — no necesita conexión a internet salvo para cargar la tipografía de Google Fonts, pero incluso sin eso se ve aceptable con fonts de respaldo.
